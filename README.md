# 🚚 DeliverAI: E-commerce Delivery Prediction System
## 📌 Project Overview
This project uses Machine Learning to predict whether an e-commerce product will be delivered on time or not. It helps improve logistics, customer satisfaction, and operational efficiency.

## 🎯 Objectives

* Predict delivery status (On Time / Late)
* Identify key factors affecting delivery performance
* Improve logistics planning
* 
## 📊 Dataset

* File: `E_Commerce.csv`
* Features include:

  * Shipment Mode
  * Customer Care Calls
  * Discount Offered
  * Product Importance
* Target Variable:

  * `Reached.on.Time_Y.N` (1 = On Time, 0 = Late)

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib & Seaborn
* Scikit-learn
* 
## 🧠 Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest ⭐ (Best Performance)
* K-Nearest Neighbors (KNN)

## 📈 Key Insights

* Higher discounts often lead to late deliveries
* More customer care calls indicate delivery issues
* Product importance affects delivery priority

## 🚀 How to Run

```bash
pip install -r requirements.txt
```

Open Jupyter Notebook:

```bash
jupyter notebook delivery_prediction.ipynb
```

## 📌 Results

* Random Forest achieved the best accuracy
* Model successfully predicts delivery status

## 🙌 Conclusion
Machine learning can significantly improve delivery prediction and logistics planning in e-commerce systems.
